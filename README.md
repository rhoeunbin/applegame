# 🍎 사과 게임 (Apple Game)

## 📝 프로젝트 개요

'사과 게임'은 1부터 9까지의 숫자가 무작위로 배치된 보드에서, 플레이어가 여러 숫자를 조합하여 합이 10이 되도록 만드는 캐주얼 퍼즐 게임입니다. 제한 시간 1분 안에 최대한 많은 사과를 없애고 높은 점수를 획득하는 것이 목표입니다.

## 🎮 게임 방법

1.  **게임 시작:** '새 게임 시작' 버튼을 클릭하거나 페이지를 로드하면 게임이 시작됩니다. 1분 타이머가 작동하며, 보드에 1부터 9까지의 숫자가 무작위로 채워집니다.
2.  **사과 선택 (드래그):**
    *   마우스를 누른 채로 사과(숫자) 위에서 드래그를 시작합니다.
    *   마우스를 떼지 않고 인접한(상하좌우, 대각선) 다른 사과 위로 이동하면 해당 사과가 선택 그룹에 추가됩니다.
    *   **연결 규칙:** 다음 사과를 선택할 때, 직전 사과와 다음 사과가 다음 조건 중 하나를 만족해야 합니다.
        *   **바로 인접:** 두 사과가 상하좌우 또는 대각선으로 바로 붙어있는 경우.
        *   **빈칸을 가로지르는 직선:** 두 사과가 같은 가로줄 또는 세로줄에 있으며, 그 사이에 있는 모든 칸이 비어있는(사라진) 경우.
3.  **합계 10 만들기:**
    *   드래그하는 동안 선택된 사과들의 숫자 합이 실시간으로 계산됩니다.
    *   **합이 정확히 10이 되면:** 선택되었던 모든 사과들이 보드에서 사라지고, 점수 10점을 획득합니다. 이후 선택 상태는 초기화됩니다.
    *   **합이 10을 초과하면:** 잘못된 조합으로 간주되어, 현재까지의 모든 선택이 즉시 취소되고 초기화됩니다.
4.  **선택 취소:** 합이 10이 되지 않은 상태에서 마우스 버튼을 떼면, 현재까지 선택했던 모든 사과들의 선택이 취소됩니다.
5.  **게임 종료:**
    *   제한 시간 1분(60초)이 모두 소진되면 게임이 종료됩니다.
    *   보드의 모든 사과를 성공적으로 없애면 게임이 종료됩니다.
    *   게임 종료 시 최종 점수가 표시됩니다.

## ✨ 주요 기능

*   **직관적인 드래그 선택:** 마우스를 이용한 연속 드래그 방식으로 사과를 선택하여 합이 10이 되는 조합을 만듭니다.
*   **확장된 연결 규칙:** 인접한 사과뿐만 아니라, 빈칸을 가로지르는 직선상의 사과도 연결하여 없앨 수 있습니다.
*   **시간 제한:** 1분이라는 제한 시간으로 게임의 긴장감을 더합니다.
*   **점수 시스템:** 없앤 사과 조합에 따라 점수를 획득하고, 최종 점수를 확인할 수 있습니다.
*   **반응형 디자인:** 다양한 화면 크기에서 플레이할 수 있도록 최적화된 디자인을 제공합니다.
*   **귀여운 사과 이미지:** 시각적으로 즐거운 게임 경험을 제공합니다.

## 🛠️ 사용 기술

*   **HTML5:** 게임의 구조와 콘텐츠를 정의합니다.
*   **CSS3:** 게임의 시각적인 스타일링과 레이아웃을 담당합니다. (Bootstrap 4.5.2 활용)
*   **JavaScript (ES6+):** 게임의 핵심 로직, 상호작용 및 동적인 요소를 구현합니다.

## 🚀 설치 및 실행

1.  **프로젝트 클론 (또는 다운로드):**
    ```bash
    git clone [프로젝트_저장소_URL] # GitHub에 올렸을 경우
    # 또는 이 파일을 포함한 apple-game 폴더를 다운로드합니다.
    ```
2.  **디렉토리 이동:**
    ```bash
    cd apple-game
    ```
3.  **게임 실행:**
    *   `index.html` 파일을 웹 브라우저(Chrome, Firefox 등)로 드래그 앤 드롭하거나, 파일을 더블 클릭하여 엽니다.
    *   또는, 로컬 웹 서버를 사용하여 실행할 수 있습니다 (예: Python이 설치되어 있다면 `python3 -m http.server` 명령어를 `apple-game` 디렉토리에서 실행 후 `http://localhost:8000` 접속).
